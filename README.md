# molipe_01

Welcome Molipe (Modular Live Performer) - A modular environment for live improvisation in electronic music. Built with Pure Data, it runs on a Raspberry Pi 5 with a dedicated controller — but it’s fully cross-platform and works on any operating system.

Please download and install Pure Data (Vanilla) to run Molipe: 
https://puredata.info/downloads/pure-data

To start Molipe please open: mother.pd - from within mother.pd you can load different patches.
